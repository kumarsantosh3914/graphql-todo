# GraphQL TODO Project

This is a GraphQL TODO application built with Express and TypeScript, featuring Apollo Server for handling GraphQL queries and MongoDB as the database for persistence.

## Features

- 🚀 **GraphQL API**: Built with Apollo Server for a modern, efficient API.
- 🗄️ **MongoDB Integration**: Persistent data storage with Mongoose ODM.
- 📝 **TODO Management**: Full CRUD operations for managing tasks.
- 🔐 **Type-Safe**: Leveraging TypeScript for robust, maintainable code.
- 🎯 **Express Framework**: Lightweight and flexible web application framework.
- 📊 **Data Validation**: Zod validation for request/response schemas.
- 📋 **Logging**: Winston for comprehensive application logging with MongoDB support.

## Prerequisites

Before you begin, ensure you have the following installed on your system:

- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [MongoDB](https://www.mongodb.com/) (local or cloud instance)
